# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within SuperOC (e.g., state leakage, arbitrary code execution via markdown injection, or lock bypassing), please do not open a public issue.

Instead, please report via GitHub Security Advisories:
https://github.com/muntasirrmahdi/superoc/security/advisories

We will acknowledge receipt within 48 hours and work with you to remediate the issue safely before public disclosure.
