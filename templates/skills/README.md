# Skills Template

Create your own skills in this folder. Follow the format:

```
~/.superoc/skills/
|-- <skill-name>/
|   `-- SKILL.md
```

## Example SKILL.md

```markdown
# Skill Name

Description of what this skill does.

## Triggers
- "do X"
- "when Y happens"

## Actions
1. Step 1
2. Step 2
```

## Quick Start

1. Create folder: mkdir -p ~/.superoc/skills/my-skill
2. Add SKILL.md with your skill definition
3. Register in skills-registry.json
